# from the CUDA (cu121) or CPU PyTorch channel, which this generic index can't

#

# derived by walking the import graph from server.py's entry points INCLUDING
# the modules Hydra instantiates from configs/firefly_gan_vq.yaml (spectrogram,

# pyproject pulls (funasr, faster_whisper, datasets, gradio, tensorboard,
# modelscope, silero-vad, pyaudio, wandb, ...) are deliberately OMITTED -
# inference never imports them.  wandb is referenced only inside an unused
# training helper (lazy import), so it is not required.
# ============================================================================
